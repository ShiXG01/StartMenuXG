# StartMenuXG

`StartMenuXG` 是一个面向 Windows 的桌面启动菜单工具。  
它可以通过全局热键快速呼出，用磁贴方式展示菜单项，并支持搜索、拖拽排序、虚拟分类和背景图自定义。

---

## 快速开始

1. 运行 `StartMenuXG.exe`
2. 第一次使用时，先在设置里选择你的菜单目录
3. 默认使用 `Win+Z` 呼出或收起菜单
4. 把常用的文件、文件夹、快捷方式放进菜单目录

---

## 核心功能

### 1. 全局热键呼出

- 默认热键：`Win+Z`
- 支持在设置中切换预设热键
- 再次按热键可直接收起菜单

### 2. 磁贴式菜单

- 菜单项以磁贴方式显示
- 图标在上，名称在下
- 支持手动拖拽磁贴排序
- 磁贴会按网格吸附

### 3. 虚拟分类

- 可以直接在菜单里新建分类
- 支持重命名分类
- 支持删除分类
- 支持收起 / 展开分类
- 支持把磁贴拖进分类
- 支持把磁贴拖回 `未分类`
- 支持拖动分类头调整分类顺序
- `未分类` 固定显示在底部

说明：

- 这里的分类是“虚拟分类”
- 不会改变你真实菜单目录中的文件结构
- 删除分类时，不会删除真实文件，只会把其中的磁贴退回 `未分类`

### 4. 搜索与键盘操作

- 顶部搜索框可过滤当前已加载的菜单项
- `Enter` 打开当前选中项
- `Esc` 关闭菜单
- 支持方向键切换选中项
- `F5` 刷新菜单

### 5. 窗口与主题

- 无边框圆角窗口
- 跟随 Windows 浅色 / 深色主题
- 支持背景图
- 支持透明度调整
- 菜单窗口只允许通过顶部栏拖动

### 6. 自动收起

- 点击菜单外部可关闭
- 菜单失去焦点可自动关闭
- 按 `Esc` 可关闭

---

## 菜单目录规则

当前菜单扫描规则如下：

- 只扫描菜单目录当前层
- 不递归扫描子目录内部文件
- 支持文件、文件夹、`.lnk`、`.url`

这意味着：

- 适合做“常用启动项面板”
- 不适合拿来做深层目录浏览器

## 配置文件

程序会在 `data` 目录下保存配置。

### `data/settings.json`

保存基础设置，例如：

- 菜单目录
- 背景图片
- 热键预设
- 透明度
- 字体大小
- 窗口尺寸
- 图标大小

### `data/menu_layout.json`

保存菜单布局，例如：

- 磁贴排序
- 分类信息
- 分类收起状态
- 分类内项目归属
- 分类显示顺序

如果你想重置分类和磁贴布局，可以在关闭程序后删除这个文件。

---

## 开机自启

如果需要开机自启，可以把 `StartMenuXG.exe` 的快捷方式放进 Windows 当前用户的启动目录。

操作方法：

1. 按 `Win + R`
2. 输入 `shell:startup`
3. 回车
4. 把 `StartMenuXG.exe` 的快捷方式放进去

## 从源码运行

如果你要从源码运行当前项目：

```powershell
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python main.py
```

## 重新打包

当前项目使用 `PyInstaller` 打包。

示例：

```powershell
.venv\Scripts\python.exe -m PyInstaller --clean --noconfirm StartMenuXG.spec
```

---

## 当前限制

- 只支持 Windows
- 菜单项扫描是非递归的
- 分类是虚拟分类，不会同步改动真实目录结构
- 第一次启动时如果菜单目录为空，界面不会自动生成示例内容
