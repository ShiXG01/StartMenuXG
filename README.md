# StartMenuXG

`StartMenuXG` 是一个基于 `Python + PySide6` 开发的 Windows 桌面启动菜单程序，用来快速呼出菜单、搜索当前菜单项，并打开文件、文件夹或快捷方式。

---

## 功能概览

### 1. 全局热键呼出

- 默认使用 `Win+Z` 呼出或收起菜单
- 支持在设置中切换预设热键
- 当前内置预设：
  - `Win+Z`
  - `Ctrl+Alt+Z`
  - `Ctrl+Shift+Z`
  - `Alt+Z`

### 2. 菜单加载

- 从指定菜单目录读取项目
- 支持显示：
  - 文件
  - 文件夹
  - `.lnk` 快捷方式
  - `.url` 快捷方式
- 当前菜单扫描为非递归模式
  - 只读取菜单目录当前层
  - 不递归扫描子目录内部文件

### 3. 搜索过滤

- 顶部提供搜索栏
- 只搜索当前已经加载到菜单中的项目
- 文件按文件名搜索
- 文件夹按目录名搜索

### 4. 键盘操作

- `Enter` 打开当前选中项
- `Esc` 收起菜单
- 搜索框有焦点时，可直接使用上下方向键切换选中项
- `F5` 刷新菜单


### 5. 界面样式

- 无边框窗口
- 圆角面板
- 跟随 Windows 浅色 / 深色主题
- 主面板支持半透明显示
- 搜索框、列表区、底部按钮带透明或半透明效果

### 6. 背景图支持

- 可在设置中选择背景图片
- 背景图会铺满主面板并自动裁切适配
- 背景图上会叠加遮罩，保证文字可读性

### 7. 设置持久化

配置会保存到：

- `data/settings.json`

程序重启后会继续使用上次保存的设置。

---

## 当前可配置项

设置窗口当前支持以下内容：

- 菜单目录
- 背景图片路径
- 全局热键预设
- 面板透明度
- 字体大小

程序内部还会持久化以下参数：

- 窗口宽度
- 窗口高度
- 图标大小
- 屏幕边缘安全边距

---

## 主界面操作

主界面底部提供以下按钮：

- `打开目录`
  - 打开当前菜单目录
- `刷新`
  - 重新扫描当前菜单目录
- `设置`
  - 打开设置窗口
- `打开项目`
  - 打开程序项目目录
- `退出`
  - 关闭程序

## 开机自启

当前项目已经可以通过 Windows 当前用户的 Startup 启动目录实现开机自启。

### 手动设置流程

1. 按 `Win + R`
2. 输入 `shell:startup`
3. 回车，打开当前用户的启动文件夹
4. 把 `StartMenuXG.exe` 的快捷方式，或者一个用于启动它的脚本，放进这个目录

### 关闭开机自启

如果以后不需要开机自启，直接删除下面这个文件即可：

- `C:\Users\Administrator\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup\StartMenuXG.vbs`

### 说明

程序当前是“启动后显示主窗口”的行为，不是静默后台驻留。

这意味着登录 Windows 后如果自启生效：

- 程序会自动启动
- 主窗口也会跟着出现
